telnetlib3-client TUI Help
=========================

The ``telnetlib3-client`` TUI provides interactive editors for macros,
autoreplies, and highlights.  Each editor includes a built-in help dialog
describing its features, keyboard shortcuts, and the command syntax used
by macro and autoreply text fields.
